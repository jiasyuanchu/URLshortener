# URL Shortener 短網址產生器

## 簡介
#### 把長網址縮短的小工具

## 功能
- 在上方空格輸入你要縮短的網址
- 按下右方的「Shorten」按鈕
- 出現縮短網址，按「Copy」按鈕複製網址
- 可透過新產生的短網址連結到原本網址

## 開始使用

1. 安裝node.js及npm
2. 將專案 clone 到本地
3. 在本地開啟後，透過終端機輸入
  ```javascript
  npm install
  ```
4. 安裝完畢後，繼續輸入：
  ```javascript
  npm run start
  ```
5. 打開瀏覽器進入到以下網址，若看見以下此行訊息代表正常運作
  ```javascript
  Listening on http://localhost:3000
  ```

6. 暫時停止使用時，請輸入
  ```javascript
  ctrl + c
  ```
